Chapter 41: Palisade

Sie bekommen ein Gitter an Quadraten, von denen einige Ziffern enthalten. Ihre Aufgabe besteht darin, die Gitter in gleichflächige Regions zu teilen, so dass jedes nummeriertes Quadrat neben so viele Grenzen ist (einschließlich die zwischen der Innenseite und der Außenseite des Gitters).

Dank für dieses Puzzle geht an Nikoli, die es ‘Five Cells’ nannten. [22].

Palisade wurde von Jonas Kölker zu dieser Sammlung beigesteuert.

[22] http://nikoli.co.jp/en/puzzles/five_cells.html

41.1 Palisade-Steuerung

Ein Linksklick legt eine Grenze. Ein Rechtsklick markiert ‘keine Grenze’. Verwenden Sie alternativ die Pfeiltasten, um den Positionsanzeiger um den Gitter zu bewegen. Halten Sie die Strg-Taste und drücken Sie eine Pfeiltaste um eine Grenze zu legen. Halten Sie die Umschalt-Taste und drücken Sie eine Pfeiltaste um eine Grenze zu lösen. Wiederholen Sie so einen Tastendruck, um seinen Gegenteil durchzuführen.

(Alle in section 2.1 beschriebene Aktionen sind auch verfügbar.)

41.2 Palisade-Parameter

Diese Parameter sind von der Option ‘Custom...’ im Menü ‘Type’ verfügbar.

Width (Breite), Height (Höhe)
Größe des Gitters in Quadraten
Regionsgröße
Die Größe der Bereiche, in die das Gitter unterteilt werden müssen.
